Using C++ to implement PSO


Type the following command to compile the program:
```cpp
make
```
Type the following command to run the program:
```cpp
./PSO [Run] [Dimension] [Function_number] [Population] 
```



For example
```cpp
./PSO 1 10 1 50 
```
